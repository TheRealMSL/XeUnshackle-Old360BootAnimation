# How to swap success.wmv video?

## Tools required:

To swap the success.wmv file to your own choice, you will need:
* Microsoft Expression Encoder 4
  * This was tested on Windows 7, you can attempt to use Windows Media Encoder 9 but this guide won't be covering that program.
* Your source video file
  * MUST be in .avi, .mpg, .wmv. This program will not accept .mp4 files. You can use ffmpeg or online-convert to convert the source video file to the required format.
* Build environment
  * See BUILD.MD

## Encoding format
* Import your video 
  * Go to Edit > Apply Preset > Encoding For Devices > WMV > Either select VC-1 Xbox 360 720p or 1080p
* After preset is selected, press CTRL + E to encode
* By default, the encoded file will be located in your documents folder called "Expression" and will be in the folder "Output"
